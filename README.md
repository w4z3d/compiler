## How to build

- [Linux](#linux)
- [Windows](#windows)

### Linux

```bash
cd build && cmake .. -DCAMKE_BUILD_TYPE=Release && cmake --build . --parallel 8
```

### Windows

```bash
cd build && cmake .. -DCAMKE_BUILD_TYPE=Release && cmake --build . --parallel 8
```
